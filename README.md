# 𓁹‿𓁹 Sistem Management Kos - Anya Theme 🥜

Aplikasi Web Management Kos berbasis **Flask (Python)** dengan antarmuka bertema manis khas **Anya Forger (`Spy x Family`)**.

---

## 🎨 Fitur Utama

- **Authentication System**: Login & Logout khusus Admin Kos.
- **Dashboard Overview**: Ringkasan total penghuni kos, total pemasukan, dan tabel transaksi pembayaran terbaru.
- **Data Penghuni**: Pengelolaan data penghuni kos.
- **Data Pembayaran**: Pencatatan dan pemantauan status pembayaran bulanan.
- **Aesthetic UI**: Desain responsif Bootstrap 5 dipadu dengan *custom CSS* pastel bertema Anya Forger.

---

## 🛠️ Teknologi yang Digunakan

- **Backend**: Python 3, Flask
- **Frontend**: HTML5, CSS3, Bootstrap 5, Jinja2 Templates
- **Version Control**: Git & GitHub
- **Deployment**: PythonAnywhere
